# Calculadora
## Descripción
Una calculadora común, muy simple, permite hacer las operaciones mas comunes de manera sencilla.
## Requisitos
* Sistema Operativo Android Lollipop 5.1 (minimo)
## Como Usar?
1. Apretar los botones para ingresar un número
2. Ingresar algún operador
3. Ingresar otro número
4. Apretar "=" y estará su resultado.
## Realizado con
* Android Studio 3.3
* Zelezny
* ButterKnife 9.0rc1
* MathJS Android 0.2.1
* JDK 8
